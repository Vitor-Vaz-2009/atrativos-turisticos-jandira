# Atrativos Turísticos de Jandira

Projeto desenvolvido em HTML e CSS com o objetivo de apresentar alguns atrativos turísticos da cidade de Jandira.

## 📌 Sobre o projeto

O site mostra pontos turísticos e locais conhecidos da cidade utilizando um layout em colunas, imagens e descrições.

## 🖼️ Atrativos apresentados

- Parque Ecológico de Jandira
- Teatro Municipal de Jandira
- Praça de Eventos Elias Jaburd
- Letreiro Jandira SP
- Parque Cidade da Criança
- Boulevard de Jandira

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
